Требования ко второму этапу - Функция поиска в бинарном файле вхождений последовательности символов в файле.

Требования к основной форме.
На pSearchFiles добавить компонент для ввода последовательностей. Выбор файла через TFileOpenDialog с отражением в поле edtStartFolder.

Требования к SearchFile.dll, 
Добавим Функцию поиска в бинарном файле вхождений последовательности символов в файле (SearchInFile).  
Ожидаемый результат - количество найденных вхождений. 
Например, (“libsec”, “D:\temp\BigFile.bin”). 
Здесь “libsec”- однобайтовые символы.

Дополнительно:  
• возможность 	поиска 	одновременно 	нескольких 	последовательностей символов (“libsec”, “binsec”). 
• возврат списка позиций найденных вхождений для каждого из искомой последвательности. 
• Должна иметься опция прерывания выбранной задачи.
• Должен отображаться ход выполнения текущих задач в реальном времени.
