Необходимо создать модульное приложение для Windows mkos.exe на языке Delphi с функционалом разделённым по dll.

Первая DLL (SearchFile.dll):
1 задача - функция (SearchFiles) рекурсивного поиска списка файлов по маске и стартовой папке поиска. Ожидаемый результат поиска файлов - количество найденных файлов, например, входные параметры: маски, например, (“*.pas”; “*.dfm”, ...), стартовый каталог, например, “D:\Develop\”. Дополнительно: возврат списка полных путей найденных файлов.
2 задача - Функция (SearchInFile) поиска вхождений последовательности символов в одном из выбранных файлов. Ожидаемый результат - количество найденных вхождений. Например, (“libsec”, “p:\Dumps\BigFile.bin”). Здесь “libsec”- однобайтовые символы, поиск в бинарном файле.  Возможность поиска одновременно нескольких последовательностей символов (“libsec”, “binsec”). Возврат списка позиций найденных вхождений для каждого из искомых слов. 

Вторая DLL (Arch7zip.dll) - процедура или функция выполняющая архивирование файлов указанного каталога с помощью программы 7-zip.

Вызывающая программа не должна зависать, при запуске продолжительной по времени выполнения задачи. 

Основной исполняемый файл (.exe) интерфейс вызова задач. Список задач (всего 3) и их параметры жестко заданы в основном файле.

Ожидается, что:
• задачи будут запускаться асинхронно не приводя к зависанию программы;
• будет реализована динамическая загрузка .dll.

Интерфейсное окно:
• Должно отображать список доступных для запуска задач.
• Должно давать возможность запустить любую доступную задачу с необходимым для команды набором параметров. Важно - выполняться задачи должны асинхронно.
• Должно отображать список текущих выполняемых задач и уже выполненных. 
• Должно давать возможность просмотреть результаты выполнения по каждой задаче.

Дополнительно:

• Должна иметься опция прерывания выбранной задачи.
• Должен отображаться ход выполнения текущих задач в реальном времени.

В основном окне должны задаваться стартовая папка и маски для поиска.
Для поиска файлов использовать TDirectory.GetFiles.
Ввод масок через ТMemo.
Выбор стартового каталога через TOpenTextFileDialog, отображение в TEdit.
Для передачи строковых параметров между основным приложением и DLL использовать PChar.
 
Задачи реализуем по этапам.
На данном этапе реализуем основной файл (Exe) и первую задачу из первой dll (поиск файлов).
Использовать версию Delphi 11.
